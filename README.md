# Jingles

The Ultimate Video Game Jingle Repo for Cocoon.

This repository currently contains **2,882 jingles** and started as a large Nintendo-focused collection before expanding into a full multi-platform library.

Nintendo is still the core of the repo, but `v2.0` adds a massive batch of new jingles from Sega, PlayStation, Dreamcast, Saturn, Steam, and several smaller niche platforms as well.

## Adding the repo to Cocoon

Add this repository through `Cocoon Settings -> Jingle Settings -> Add Repository`.

After that, you can individually choose which jingle you want per game.

## Matching

The repo includes a proper `index.json` with regex matching, so Cocoon has a much better chance of finding the correct jingle even when game names are written differently.

## Goal

The goal is to keep pushing coverage as far as possible across both Nintendo and non-Nintendo platforms, while also improving match quality and replacing entries with cleaner versions when better sources are available.

## Contributions

If you know missing jingles, find wrong matches, notice missing games, or have cleaner versions of existing jingles, feel free to reach out or open a contribution.

The aim is for this repo to become a go-to jingle library for Cocoon instead of another half-finished collection.
